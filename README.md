# Prescott Girls website (static)

This is a clean, traditional, responsive static website template for **The Prescott Girls**.

## What’s included
- 7 pages: Home, Book, Author, Illustrations, Teachers, Events, Contact
- Simple CSS (no build tools)
- SVG placeholder images in `/assets` you can replace

## Quick customization
1. Replace placeholder images:
   - `assets/book-cover.jpg` (book cover)
   - `assets/author-photo.jpg` (author photo)
   - `assets/illustration-1.svg` etc.

   You can replace SVGs with JPG/PNG/WebP using the same filenames.

2. Update text in each `.html` file.
3. Update the email address in `contact.html`.

## Fonts
CSS prefers **Caslon** where available:
`"Adobe Caslon Pro", "Big Caslon", "Caslon" ...`
If you want guaranteed Caslon-like rendering on all devices, consider switching to a web font such as *Libre Caslon Text* (Google Fonts) and add it in the `<head>`.

## Hosting
- Any basic web host works.
- GitHub Pages works well (upload the folder to a repo and enable Pages).
- Netlify / Cloudflare Pages also work (drag-and-drop deploy).

## Notes
This site is intentionally simple and readable. If you want:
- Mailing list signup
- Form handling
- Blog engine
you can add those later (or integrate a service).
